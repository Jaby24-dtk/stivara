# Stivara

Corporate secretarial SaaS. Source of truth for product scope is `Stivara-Build-Brief.md` (kept outside this repo — ask the user if you need it re-shared). This repo currently implements Phase 0 only: auth, company/person/role data model, document management, Singapore-only compliance calendar, AI document search (RAG), basic tasks.

Do not build Phase 1+ features (registers, resolutions/minutes, meetings, filings, other jurisdictions, client portal, invoicing, e-signatures) without confirming scope first — see the brief's Section 12 phasing.
